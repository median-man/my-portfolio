codeDeck

codeDeck is an educational flash card game and study tool. Currently, codeDeck offers a single deck of flash cards to review Git commands and concepts. The application was built in a little over a week by a group of four developers. The application was built with Google Firebase, jQuery, and Bootstrap 3. Additional information is presented to users via the Wikipedia API.

  • Live Demo: codeDeck
  • Date: September 2017
  • Code: Git Hub

Train Scheduler

Train Scheduler is a real-time database app where a user can add and remove data stored in a Google Firebase Database. When data is added or removed in the database, the changes are reflected on the page in real time. The application was built using Google Firebase, Twitter Bootstrap v3, jQuery, and MomentJS.

  • Live Demo: Train Scheduler
  • Date: September 2017
  • Code: Git Hub

Board Game Trivia

Board Game Trivia challenges the player to answer all the questions before time runs out. After the quiz is finished, the score is displayed. This project was built with jQuery, HTML5, and Bootstrap 3 CSS. The styling is mobile responsive.

  • Live Demo: Board Game Trivia
  • Date: August 2017
  • Code: Git Hub

Hangman: National Parks

National Parks themed Hangman game that is fully mobile responsive built with Javascript, HTML5, and Bootstrap 3 CSS. An image related to the park is displayed if you can solve the word by guessing the correct letters.

  • Live Demo: Hangman: National Parks
  • Date: August 2017
  • Code: Git Hub

Gif-Tastic Gallery

This mobile responsive web page displays Gif images by requesting them via the Giphy API based on pre-selected topics or topics entered by the user. The application was built with jQuery, HTML5, CSS3, and the Giphy API.

  • Live Demo: Gif-Tastic Gallery
  • Date: August 2017
  • Code: Git Hub

Eat-Da-Burger

A user can view a list of hamburgers and mark them if they have been eaten. The application maintains a MySql database to save the state of the burgers. Users may add more burgers to the list and the entries are added to the database. This application was built using the Express framework for Node and MySql on the backend. Handlebars was used to create semantic templates for rendering the HTML.

  • Live Demo: Eat-Da-Burger
  • Date: September 2017
  • Code: Git Hub
